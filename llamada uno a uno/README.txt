para ejecutar servidor
>>python serverF.py [port]

para ejecutar cliente:
>>python clientF.py [ipserver:port] [ipcliente] [portcliente] [nickname]

Comandos:

>>login  #para logearse y agregar el socket al diccionario
>>listar #muestra la lista de usuarios conectados

>>audio [nickname_destino] #enviar audios 

>>hablar [nickname_destino] #permite hablar continuamente a un cliente 
#se debe ejecutar esta opcion en los dos clientes para establecer comunicacion.
